# 中文版本

项目简介
地图是一种科学表达的方式，有着非常悠久的历史，从古代到近现代，地图与科技的融合是渐进而又显著的。出于对中国文化的热爱，我们把古代是人们走过的地方做成了3维地图生动地展示了诗人的足迹，并对各诗人的生平进行了调研，丰富了展示内容。
